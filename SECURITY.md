# Security Policy

## Supported Versions

Only the latest version of golangci-lint is supported.

We will only release security updates for the latest version.

## Reporting a Vulnerability

We want to keep our software safe for everyone.

If you've discovered a security vulnerability in golangci-lint, we appreciate your help in disclosing it to us in a responsible manner, by creating a [security advisor](https://github.com/beltranaceves/uber-go-lint-style/security/advisories).

For us to respond to your report most effectively, please include any of the following:

Steps to reproduce or a proof-of-concept
Any relevant information, including the versions used
